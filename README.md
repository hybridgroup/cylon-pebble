# Cylon.js For Pebble

Cylon.js (http://cylonjs.com) is a JavaScript framework for robotics and physical computing using Node.js

This module provides an adaptor and driver for the Pebble smart watch (http://getpebble.com/). It uses the Pebble 2.0 SDK, and requires the 2.0 iOS or Android app, and that the the "Chomps" app has been installed on the Pebble watch.

[![Build Status](https://secure.travis-ci.org/hybridgroup/cylon-pebble.png?branch=master)](http://travis-ci.org/hybridgroup/cylon-pebble)

## Getting Started
Install the module with: `npm install cylon-pebble`

## Examples

```javascript
var Cylon = require('cylon');

...
```

```coffee-script
Cylon = require 'cylon'

Cylon.robot
...
```
